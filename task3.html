CSS

CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the presentation of an HTML document. CSS is used to define the layout, colors, fonts, and other visual aspects of a web page. It is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.
